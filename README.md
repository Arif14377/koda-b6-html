## Fitur Aplikasi
### Authentication Flow:
1. Register, menyimpan data ke local storage.
2. Login, cek validasi email & password dari local storage.
3. lupa password, cek validasi email di local storage.

### Homepage
1. Menampilkan data product yang diambil (fetch) dari data json.
2. menampilkan data testimoni dari data json
3. ketika klik buy product, diarahkan ke detail product tersebut.

### payment details
